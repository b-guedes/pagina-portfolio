Projetos

Pipeline de Dados

Projeto EBAC

Desenvolvi um pipeline de dados coletados do Telegram via bot, abrangendo coleta, processamento e análise de dados (ETL) com ferramentas da AWS. O objetivo foi criar um modelo capaz de coletar dados para analisar o comportamento de usuários.

Palavras-chave: ETL, AWS, SQL, Apache Spark, AWS Lambda

Automação de Processos

Projeto Compass UOL

Desenvolvi um processo de automação em Python para extrair informações de planilhas, coletar dados de APIs e sites, e criar uma planilha final com os dados organizados. Esse processo, que manualmente demoraria cerca de uma hora, foi reduzido para apenas dois minutos com a automação.

Palavras-chave: Automação, Python, Pandas, Excel, API
